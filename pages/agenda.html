[ SESIÓN 02 ]

CODE WITNESS · Self-Healing Codebases
⏱ Duración: 30–45 min
Sesión de la comunidad · Emilio Delgado · SonarQube + MCP

CODE WITNESS — taller presencial centrado en la calidad del código que produce la IA. Combinaremos GitHub Copilot CLI, SonarQube y Model Context Protocol para auditar, consultar hallazgos y aplicar correcciones de forma asistida.

  • Detección de code smells, bugs y vulnerabilidades con SonarQube
  • Consulta automática de hallazgos mediante MCP
  • Propuesta y aplicación de refactorizaciones con Copilot CLI

Resultado: un ciclo cerrado de generación, análisis y corrección asistido por IA para mantener bases de código más limpias, seguras y mantenibles.

Repositorio de código: https://github.com/emi-dm/VibeCoding-NASA-DevDays

[ SESIÓN 03 ]

Hands-on (Python, Go) y workshop TypeScript
⏱ Franja 18:40–19:40 (sin cambiar el cierre del evento)
En paralelo: 2× hands-on 30 min (PC opc.) · 1× workshop TS 60 min (PC necesaria)

Las actividades van en paralelo (asíncronas entre sí), no en serie: en un espacio, hands-on de Python (30 min) y hands-on de Go (30 min) seguidas, con Emi y Juanma — PC opcional. A la vez, en otro espacio, el workshop de TypeScript (60 min) con Kiko — ordenador obligatorio. Eliges un track; la franja de agenda sigue siendo de una hora y el evento termina a la misma hora.

  • Hands-on Python (30 min) — Emilio Delgado (Emi), PC opcional: Copilot en Python (código, tests, chat sobre fragmentos).
  • Hands-on Go (30 min) — Juan Manuel Ruiz (Juanma), PC opcional: paquetes, tests, estilo idiomático y Copilot en el día a día en Go. Repositorio de código: github.com/neovasili/2026-copilot-dev-day.
  • Workshop TypeScript (60 min) — Francisco Hermoso (Kiko), PC necesaria: práctica en el IDE con tipado, módulos y sugerencias alineadas con el proyecto. Guía del workshop: https://copilot-dev-days.github.io/agent-lab-typescript/step.html?step=guide
